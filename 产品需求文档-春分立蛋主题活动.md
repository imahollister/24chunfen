**产品需求文档 (PRD)：“春分立蛋”主题活动**

**一、 页面顶部：视觉与品牌展示 (Visuals)**

按照页面从上到下的顺序，首屏需传达浓厚的节气氛围。

- **1.1 头图 (Header):**
- **核心文案：** “春分到，蛋儿俏”。
- **辅助文案：** 据说春分这天最容易把鸡蛋立起来，人们以此庆祝春天的来临。
- **1.2 宣传横幅 (Banner):**
- **设计要求：** 适配“春分”节气背景（如：春意盎然、绿意、立蛋场景等）。

**二、 核心互动：无门槛抽奖 (Lottery Module)**

用户进入活动页面后的第一优先级转化。

- **2.1 参与机制：**
- **参与门槛：** 零门槛参与。用户进入活动页面即可直接点击（或自动触发）参与抽奖。
- **2.2 奖品设置：**
- **奖品：** 鸡蛋 + 一句祝福语。
- **参考祝福语：** “立蛋成功，全年旺！”
- **2.3 领奖逻辑与风险提示：**
- **地址收集：** 抽中实物奖品后，页面弹出地址填写弹窗。
- **强文案提示：** “地址提交后不可修改。请务必核实，若因地址错误导致派送失败，由用户自行承担后果。”
- **发放方式：** 邮寄。

**三、 深度互动：投票与评论 (Interaction Module)**

参考 image_fe0181.png 的 UI 样式进行视觉还原。

- **3.1 互动主题：** “你觉得鸡蛋哪种做法最好吃？”
- **3.2 投票配置：**
- **选项：** 水煮蛋、溏心蛋、鸡蛋灌饼、酱香鸡蛋、我来推荐！
- **交互形式：** 投票 + 评论。
- **3.3 激励机制：**
- **奖励：** 用户完成投票并发表评论后，系统实时奖励 **1 个“节气豆”**。
- **3.4 页面氛围：**
- **动态展示：** 显示当前参与总人数（如“10.9万人参与”）。
- **实时评论区：** 循环展示其他用户的推荐意见（例如：陈子羽：潮汕牛肉粉）。

**四、 商业化配置：模块化内容 (Configurable Items)**

页面底部通过三个模块进行引流或变现。

**4.1 推荐食物 (x3)**

- **展示元素：** [图片] + [名称] + [价格]。
- **配置建议：** 重点推荐春笋、香椿等春季时令食材。

**4.2 推荐商品 (x3)**

- **展示元素：** [图片] + [名称] + [价格]。
- **配置建议：** 推荐如煮蛋器、春季户外野餐垫等关联商品。

**4.3 景点/资讯 (x3)**

- **展示元素：** [图片] + [名称]。
- **配置建议：** 推荐本市（如苏州市）赏花路线或非遗立蛋体验活动。

**五、 数据统计埋点 (Data Track)**

为了后续分析，建议埋点：

- **页面 UV/PV**：整体流量情况。
- **抽奖转化率**：进入页面并提交地址的用户占比。
- **投票偏好分布**：各鸡蛋做法的得票数统计。
- **节气豆发放数**：互动活跃度指标。
